## 本地项目推送到github

>step1:　git init
>
>step2:　git add XXX
>
>step3:　git commit -m提交信息
>
>step4:　本地项目关联github：git remote add origin git@github_private:Jrepository/XXX.git
>
>step5:　本地项目推送到github：git push -u origin master
>
>其他：本地项目取消与github的关联：git remote remove origin
